1️ - What is the difference between var, let, and const?
Ans -
var, let, and const are used to declare variables in JavaScript.
var is function-scoped and it can be redeclared and reassigned. It is also hoisted.
let is block-scoped. It can be reassigned but it cannot be redeclared in the same scope.
const is also block-scoped but it cannot be reassigned or redeclared, and it must be initialized when it is declared.

2 - What is the spread operator (...)?
Ans -
The spread operator (...) is an ES6 feature that expands arrays, objects, or strings into individual elements. It is commonly used to copy elements from an array or object and create a new array or object, or to merge multiple arrays or objects together.

3 - What is the difference between map(), filter(), and forEach()?
Ans -
map() is used to apply a function to every element of an array and create a new array with the results.
filter() is used to select elements from an array that match a specific condition and return them in a new array.
forEach() is used to loop through each element of an array or object, but it does not return a new array.

4 - What is an arrow function?
Ans -
An arrow function is a shorter way to write a function in JavaScript. It uses the => symbol instead of the function keyword. It allows writing functions in a simpler and more concise way. If the function body has only one line, it can return the value automatically without using the return keyword.

Q5 - What are template literals?
Ans -
Template literals are a way to write strings in JavaScript using backticks (``). They allow writing multi-line strings and also allow inserting expressions directly inside the string using ${} without using string concatenation.
